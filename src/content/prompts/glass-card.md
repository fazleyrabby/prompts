## Prompt

Design a minimal settings card with a glassmorphic effect. Apply:
- `backdrop-filter: blur(12px)`
- A semi-transparent border (`rgba(255, 255, 255, 0.1)`)
- A subtle radial gradient for the background
- Use JetBrains Mono for system labels
- Include toggle switches with smooth transitions

## Notes

- This prompt produces best results with raw CSS tokens.
- Keep the number of interactive elements under 4 to maintain spacing.
