## Prompt

Build an analytics dashboard layout:
- Top row: 4 stat cards (Total Users, Revenue, Conversion, Growth)
- Each card: metric value, label, percentage change indicator
- Change indicators: green up arrow or red down arrow
- Main area: large chart placeholder with aspect ratio 16:9
- Bottom row: 2 smaller chart placeholders side by side
- Dark theme with subtle card borders

## Notes

- Use CSS aspect-ratio for chart containers.
- Stat cards should use tabular-nums for number alignment.
